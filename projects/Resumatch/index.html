Resume Matcher: An AI-Powered Resume-Job Description Matching System

Summary

The Resume Matcher is a comprehensive full-stack web application that leverages advanced Natural Language Processing (NLP) and Machine Learning techniques to intelligently match resumes with job descriptions. Built using modern technologies including Django REST Framework for the backend, React with Vite for the frontend, and MongoDB Atlas for data persistence, this system provides an automated, data-driven approach to resume screening and candidate evaluation. The application employs both traditional NLP methods (TF-IDF vectorization) and state-of-the-art transformer models (Sentence-BERT) to achieve nuanced, semantic understanding of textual content, resulting in accurate matching scores and actionable insights for both job seekers and recruiters.

GitHub Repository

Approach

  • Built AI-powered resume matching platform using Django REST API, React, and MongoDB Atlas, implementing Sentence-BERT transformer model and TF-IDF vectorization achieving 85%+ semantic matching accuracy between resumes and job descriptions
  • Developed NLP pipeline with spaCy and NLTK processing PDF/DOCX files, extracting structured data (skills, experience, education) from unstructured text with 90%+ precision using custom regex patterns and named entity recognition
  • Engineered weighted scoring algorithm combining keyword similarity (25%), semantic matching (35%), skills overlap (30%), and experience alignment (10%) using cosine similarity metrics and scikit-learn for objective candidate ranking
  • Created responsive React frontend with Tailwind CSS featuring drag-and-drop uploads, real-time matching visualization with Recharts, and explainable AI results showing matching/missing skills for informed decision-making
