Automated Chicken Disease Classification

Summary

Engineered an end-to-end deep learning system to automate the detection of Coccidiosis from chicken droppings images, solving the problem of inefficient manual inspection . This involved developing a transfer learning model (VGG16) and establishing a production-grade MLOps pipeline , resulting in a deployed web application that provides instant, accurate disease classification to significantly reduce livestock losses.

GitHub Repository

Approach & Core Functionality

  • Deep Learning Model: Developed an image classification model using VGG16 transfer learning with fine-tuned custom layers to achieve high accuracy in distinguishing healthy droppings from those affected by Coccidiosis.
  • MLOps & Reproducibility: Integrated DVC (Data Version Control) to manage and track the full deep learning pipeline (data ingestion, training, evaluation), ensuring reproducibility and enabling efficient re-runs by skipping unchanged stages.
  • Modular Architecture: Implemented a generic, modular coding template with structured components (entity, config, components, pipeline), promoting organized, maintainable, and scalable project architecture.

Key Technologies

  • **Python & @dataclass:** Core programming language and defining the shared, structured state for data management.
  • **LangChain/Gemini:** Provides the Generative AI capabilities for analysis, test generation, and summarization.
  • **LangGraph:** The orchestration layer that defines the fixed, multi-step pipeline and manages the state flow between agents.
  • **Streamlit:** The user interface layer, allowing easy input of the PR URL and display of the final, formatted review report.
